The `render::RapierRenderPlugin` is a small, optional plugin you can use in order to render the content of the
physics resources. It is extremely limited at the moment and will only render cuboid and ball colliders. It operates
by adding a `PbrComponents` to each entity with a `ColliderHandleComponent` attached.

```rust
use bevy::prelude::*;
use bevy_rapier2d::render::RapierRenderPlugin;

fn main() {
    App::build()
        .add_plugins(DefaultPlugins) // Default Bevy plugins.
        .add_plugin(RapierRenderPlugin)
        .run();
}
```

In the future, this renderer will be improved to display other physics elements useful for debugging
like contacts, joints, etc.
